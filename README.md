# XLiveClient

Application to interact with Xbox LIVE and Games for Windows LIVE services

## Installation

`git clone https://github.com/davispuh/XLiveClient.git`

### Dependencies

gems:

* `savon` (required)
* `builder` (required)

install manually (`gem install`) or with

`bundle install`
